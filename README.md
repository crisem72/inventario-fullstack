# 📦 Sistema de Gestión de Inventario FullStack

Aplicación web FullStack desarrollada para la gestión de stock y ventas en comercios locales.

## 🚀 Demo del Proyecto
Sistema con autenticación, roles y panel administrativo.

## 🧩 Funcionalidades
- Login con JWT
- Roles Admin / Empleado
- Gestión de productos
- Control automático de stock
- Registro de movimientos
- Dashboard diario de ventas
- Productos por unidad y por peso

## 🛠 Tecnologías
**Frontend**
- React
- Bootstrap
- Axios
- Context API

**Backend**
- Node.js
- Express
- PostgreSQL
- Sequelize ORM
- JWT
- bcrypt

## 🏗 Arquitectura
MVC + API REST
